# 데이터 전송

1. 전달 미 보장 데이터 : 대역폭 고갈시 이러한 데이터는 생략한다. (우선 순위가 낮음)
2. 전달 보장 데이터 : 수신이 보장되어야 하며, 발신도 순차적으로 보장해야하는 데이터이다.
3. 최신 상태 데이터 : 최신 상태여야만 의미가 있는 데이터이다.
4. 특급 전달 보장 데이터 : 최우선으로 보내며, 전달 보장이 되어야 하는 데이터이다.
